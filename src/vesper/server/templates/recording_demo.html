Recording Demo

The text below indicates the original goal of this demo, but not its current state. The demo currently records locally but does not save the recorded audio. See the Javascript console for recording messages.

This demo shows how you can record audio with a web browser and stream it to a server for archival. The web browser can run either on the server or on a different computer. In either case, however, the recorded audio is archived on the server.
