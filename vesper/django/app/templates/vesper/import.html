Import Recordings

Specify the server-side path of a .wav file to import that file as a recording. Specify the server-side path of a directory to import all .wav files in that directory as recordings. For directory imports, check Recursive to recursively import files in subdirectories of the specified directory, or uncheck it to exclude such files.

Select *Copy* to copy imported recordings from the specified location into the Vesper data directory, or select *Move* to move them into that directory, deleting the originals.
