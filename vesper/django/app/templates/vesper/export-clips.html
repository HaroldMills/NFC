Export Clips

Exports clips to .wav files. The clips are specified as a set of one or more detectors, one or more station/mic combinations, and a range of dates. The sound files are written to the specified output directory on the server. The export runs as a Vesper job. [In the near future, the user will be able to choose the classes of clips to be exported. The clips will also be delivered to the client in a zip file.]
