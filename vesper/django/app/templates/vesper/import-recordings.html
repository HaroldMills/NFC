Import Recordings

Submit this form to import one or more recordings into Vesper. Each recording must comprise one or more .wav files in the Vesper Server file system. The imported files are copied into the Vesper data directory, and the original files are not modified. The import runs as a Vesper job.

Specify the paths of one or more .wav files and/or directories containing .wav files to import those files as recordings. Each path should be specified on a separate line. Multi-file recordings are automatically recognized from the stations, start times, and durations of the imported files.
