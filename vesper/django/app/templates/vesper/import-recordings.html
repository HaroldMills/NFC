Import Recordings

Imports one or more recordings into Vesper. Each recording comprises one or more .wav files in the file system of the computer on which the Vesper server is running. The server does not copy the files, but rather remembers their paths in the file system, so the files should not be moved after import. The import runs as a Vesper job.
